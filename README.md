# TaskFlowManager

## About the Project

TaskFlowManager is a simple web application that helps users manage their daily tasks. Users can add tasks, mark tasks as completed, and delete tasks dynamically without reloading the page.

## Features

- Add new tasks
- Display task list
- Mark tasks as completed
- Delete tasks
- Dynamic updates using JavaScript
- Responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

TaskFlowManager
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in any browser.
4. Add and manage tasks.
